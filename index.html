Sigma Features

  • Enhanced Python

    Sigma adds to Python new modules like os or ulab

  • Themes

    Sigma has a variety of themes to allow you to have the best experience with your numworks

  • Gameboy

    Sigma integrates natively a GameBoy emulator to allow you to run any .gb file

  • Customizable

    Sigma also integrates a ton of new settings to improve th user experience

  • Symbolic Calculation

    Symbolic computation was removed from Epsilon. Sigma reintroduces that feature.

  • Dual Boot

    Sigma uses Upsilon's bootloader, which allows you to have up to 3 partitions to boot on.

  • Protection against lock

    The custom bootloader prevents you to update your calculator towards an OS that lock your calculator

  • Plenty of other features
